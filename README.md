# HeartChat AI

A minimal **AI social chat** demo inspired by modern anime-style AI companion platforms, restyled for the **Western market** with a deep purple, neon accent theme.

This project is built with **Next.js + TypeScript** and is ready to push to **GitHub** and deploy on **Vercel**.

## Features

- **Discover homepage** with character cards, search, and tag filters
- **Chat page** with:
  - Left sidebar chat list
  - Center chat conversation with message bubbles
  - Right side profile / vibe panel
  - Simple in-browser AI reply simulation (no backend, no API key)
- **Purple first** visual style tuned for Western AI social apps (neon highlights, glass panels, dark gradient background)

## Getting Started

1. **Install dependencies**

```bash
npm install
```

2. **Run locally**

```bash
npm run dev
```

The app will be available at `http://localhost:3000`.

## Deploy to Vercel

1. Push this folder to a new **GitHub repository**.
2. Log into [Vercel](https://vercel.com) and **Import Project** from GitHub.
3. Framework preset: **Next.js** (Vercel will auto-detect).
4. Use default build command (`next build`) and output directory (`.next`).
5. Click **Deploy** and share your live URL.

No environment variables or secrets are required; everything runs fully on the client side.
